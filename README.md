# 🎮 Rock Paper Scissors Game (Python)

A simple command-line Rock Paper Scissors game with multiple difficulty levels and score tracking.

---

## 🚀 Features

- 🎯 Interactive CLI gameplay
- 🧠 3 Difficulty Levels:
  - Easy → Random moves
  - Medium → Mix of random + strategy
  - Hard → Always counters player
- 📊 Score tracking system
- 🔁 Multiple rounds support
- 🏆 Final results summary

---

## 🛠️ Requirements

- Python 3.x

No external libraries required (uses built-in modules only).
